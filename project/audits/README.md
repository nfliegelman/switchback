# project/audits

Point-in-time audits of Switchback: authorship, research integrity,
AI-residue, quality, and anything else run to check that the work
matches the owner's intent. The owner runs these periodically.

These files are snapshots, not living docs. Each is dated and never
rewritten after the fact; a later audit gets its own file rather than
editing an older one. The decision log in project/HANDOFF.md remains
the living record; an audit that produces durable decisions or deferred
work still writes them into HANDOFF.md and project/BACKLOG.md as usual.

Naming: YYYY-MM-DD-topic.md, newest by date.
